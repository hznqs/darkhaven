# DarkHaven CRM — UI QA

Use esta skill apenas para tarefas visuais e de experiência.

## Regras

- Não alterar regra de negócio.
- Não alterar backend.
- Não adicionar funcionalidade fora do escopo.
- Não mudar identidade DarkHaven.
- Não refazer design do zero.
- Corrigir apenas UI, responsividade, alinhamento, overflow, z-index, modais, drawers, tabelas, Kanban e gráficos.

## Identidade visual

- Dark premium.
- Glass morphism.
- Cards densos.
- Bordas finas.
- Tipografia premium.
- Visual SaaS moderno.
- Sem aparência genérica de IA.

## Checklist visual

Verificar:

- Elementos fora das divs.
- Cards quebrando.
- Tabelas vazando.
- Textos ultrapassando containers.
- Botões desalinhados.
- Dropdowns atrás de elementos.
- Modais/drawers cortados.
- Z-index incorreto.
- Gráficos com width/height inválidos.
- Sidebar cobrindo conteúdo.
- Overflow horizontal indesejado.

## Responsividade

Validar:

- 360px
- 390px
- 430px
- 768px
- 1366px
- 1600x900
- 1920x1080
